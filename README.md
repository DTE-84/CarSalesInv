# 🏎️ SantiCars | Automotive Discovery Engine

[![Project Status](https://img.shields.io/badge/Status-Live-success?style=for-the-badge)](https://dte-solutionsllc.gitlab.io/carsales)
[![Tech Stack](https://img.shields.io/badge/Stack-Vanilla%20JS%20|%20CSS3%20|%20GSAP-gold?style=for-the-badge)](https://dte-solutionsllc.gitlab.io/carsales)

A high-performance automotive marketplace interface optimized for complex search heuristics and high-speed inventory filtering. This project demonstrates advanced DOM manipulation and fluid UI transitions designed for commercial conversion.

---

## ⚡ Technical Highlights

- **Dynamic Inventory Engine**: Real-time filtering and sorting of vehicle assets without page reloads.
- **Fluid UI Architecture**: Implemented with high-fidelity CSS transitions and GSAP for a premium, automotive-brand feel.
- **Responsive Geometry**: Fully optimized for mobile discovery flows and desktop research sessions.
- **Optimized Asset Loading**: Asynchronous media handling to ensure zero-latency scrolling through high-resolution vehicle galleries.

---

## 🛠️ Built With

- **Frontend**: Vanilla JavaScript (ES6+), CSS3 (Custom Variables), HTML5
- **Animation**: GSAP (GreenSock Animation Platform)
- **Design Patterns**: Service-Oriented UI, Component-based CSS
- **Deployment**: GitLab Pages (Automated CI/CD)
